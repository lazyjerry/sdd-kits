# 知識庫索引

本目錄儲存**專案客製化**的注意事項，供未來任務參考。

## 使用方式

1. **任務開始前**：根據下表的「觸發時機」判斷是否需要參考相關知識
2. **任務完成後**：如有客製化的注意事項，新增知識檔案並更新此索引

**何時產生知識檔案：**
- 發現專案特有的注意事項（非一般常見知識）
- 透過用戶互動修改後發現的重要事項
- 踩過的坑或特殊處理方式

**不需要產生：** 一般程式設計知識、框架標準用法、官方文件已說明的內容

## 知識清單

| 檔案 | 觸發時機 | 摘要 |
|------|----------|------|
| `knowledge/github-blob-fetch-fallback.md` | 抓 GitHub `blob` 頁面失敗或內容不足時 | 以 `github_repo` 作為備援來源，補齊 templates/scripts/README 片段 |
| `knowledge/comparison-report-template.md` | 需要比較兩個以上專案並輸出決策報告時 | 採用固定章節模板提升一致性與可重用性 |
