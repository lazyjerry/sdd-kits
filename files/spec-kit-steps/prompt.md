# Spec Kit 提示詞範本

> 常用提示詞，直接複製貼到 Coding Agent。

```
# 建立專案治理原則
/speckit.constitution 請輸入 speckit 憲章

# 描述功能規格
/speckit.specify 請輸入規格，這裡不要輸入技術規範，以需求目的為主

# 釐清規格
/speckit.clarify

# 製作技術計畫
/speckit.plan 請輸入技術規範

# 拆解成任務
/speckit.tasks

# 分析規格與計畫
/speckit.analyze

# 開始實作
/speckit.implement

# 提醒 AI 不要超出範圍
這不在提案範圍內，請專注在 tasks.md 列出的項目
```
